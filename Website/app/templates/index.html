Tool Classifier

This classifier has been created to identify tools in a given image. The tools we currently try to recognize are a flat screwdriver, a combi wrench, a hammer, and a wrench.

Model Details

In the table below, you can find the parameters we used for our model.

The first parameter is Sequential. This is here because we use data augmentation. In each epoch, this layer will create new images from the ones it already has.

The techniques we use are random flip, random rotation and random zoom.
